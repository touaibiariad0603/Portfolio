# Abdouse Portfolio

A dependency-free, responsive portfolio website built from the three supplied projects: Dabarli, Nadra ResearchLab, and Novalign.

## Preview locally

From this folder, run:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Customize

- Personal intro and contact details: `index.html`
- Colors, typography, layout, and responsive behavior: `styles.css`
- Case-study modal content and interactions: `script.js`
- Project visuals: `assets/projects/`

## Deploy

The site is static and can be deployed directly to Vercel, Netlify, GitHub Pages, or any standard web host. No build command is required.
